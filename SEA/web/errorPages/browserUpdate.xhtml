Antes de continuar...

Esta página sólo funciona con las versiones mas recientes de los navegadores Mozilla Firefox, Google Chrome, Opera y Safari. Es posible que debas hacer unas configuraciones al navegador para poder acceder al sistema. Por favor sigue las instrucciones según tu navegador:

Mozilla Firefox

El sistema funciona a partir de la versión 52.
Si no puedes actualizar el navegador, puedes activar la bandera llamada layout.css.grid.enabled. Para ello debes escribir about:config en la barra de direcciones; recomendamos hacer esto en Firefox 45 o superior.
Dercargar Mozilla Firefox

Google Chrome

El sistema funciona a partir de la versión 57.
Si no puedes actualizar el navegador, puedes activar la bandera llamada Funciones experimentales de plataforma Web. Para ello debes escribir chrome://flags/#enable-experimental-web-platform-features en la barra de direcciones; recomendamos hacer esto en Chrome 50 o superior.
Dercargar Google Chrome

Opera

El sistema funciona a partir de la versión 44.
Si no puedes actualizar el navegador, puedes activar la bandera llamada Funciones experimentales de plataforma Web. Para ello debes escribir opera://flags/#enable-experimental-web-platform-features en la barra de direcciones; recomendamos hacer esto en Chrome 40 o superior. Aun así, no aseguramos que el sistema funcione correctamente.
Dercargar Opera Browser

Safari

El sistema funciona a partir de la versión 10.1.
Aun así, no aseguramos que el sistema funcione correctamente.
Dercargar Safari

Internet Explorer

Debido a que este navegador ya no recibe actualizaciones, no se dará soporte para este navegador.

Microsoft Edge

Lamentablemente Microsoft Edge no tiene soporte para las características que usa el sistema; aun así, estaremos pendientes de las actualizaciones de este navegador para darle soporte en futuro.
