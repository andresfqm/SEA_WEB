Cargue masivo de datos

Por favor, siga las siguientes instrucciones para realizar el cargue masivo de datos:
  • done Seleccione la tabla a la cual desea cargar/actualizar datos en el campo "tabla".
  • file_download Descargue la plantilla que el sistema le indica
  • file_upload Diligencie la plantilla y seleccione el archivo correspondiente a la tabla seleccionada en el campo "Seleccione un archivo". Posteriormente haga click en el botón "cargar".
  • error_outline Tenga en cuenta que si el sistema encuentra errores, este no cargará en la base de datos dicho registro y que de encontrar un registro duplicado, el sistema lo actualizará en la base de datos.
  • warning El sistema solo reconoce archivos con la extensión *.csv separado por comas (,), por favor asegúrese que el archivo que va a cargar cumpla con esta especificación. De no ser así, el sistema no cargará ningún registro a la base de datos.
